# 📊 ChatGPT Review Analysis

## 🔍 Project Overview
This project analyzes ChatGPT user reviews to understand customer sentiment, identify key strengths, and uncover areas for improvement using NLP techniques.

---

## 🎯 Objectives
- Analyze overall user sentiment
- Measure subjectivity of reviews
- Identify commonly praised features
- Detect major issues from negative feedback

---

## 📁 Dataset
- Total Reviews: ~147,000
- Features:
  - Review ID
  - Review Text
  - Ratings
  - Review Date

---

## 🛠️ Data Preparation
- Standardized column names
- Handled missing values
- Converted date formats
- Ensured correct data types

---

## 🧠 Sentiment Analysis
- Used TextBlob for:
  - Polarity (Positive/Negative)
  - Subjectivity
- Categorized reviews:
  - Positive (> 0.1)
  - Neutral (-0.1 to 0.1)
  - Negative (< -0.1)

---

## 📊 Key Insights

### ✅ Overall Sentiment
- Majority of reviews are **positive**
- Indicates high user satisfaction

### 📈 Subjectivity
- Average subjectivity ≈ 0.43
- Reviews are more factual than emotional

### ⭐ Ratings vs Sentiment
- Strong correlation between ratings and sentiment

### 🌟 Positive Feedback
- Common phrases:
  - "very good"
  - "best app"
  - "very helpful"

### ⚠️ Negative Feedback
Main issues identified:
- Features & functionality
- Performance & bugs
- Accuracy concerns

### 📅 Trend Analysis
- Increasing reviews over time
- Improvement in ratings and sentiment

---

## 📌 Conclusion
- Users are highly satisfied overall
- Improvements needed in features, performance, and accuracy
- Neutral users present opportunity for conversion
